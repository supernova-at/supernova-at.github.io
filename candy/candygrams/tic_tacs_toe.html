This Week's Candy

This week's Candy, should you choose to eat it, is Tic Tacs Toe!

Here is the problem statement:

You are given a string of nine (9) characters, representing a game of Tic Tac Toe. Write an algorithm to evaluate the board. The possible outcomes are:

  • X wins
  • O wins
  • Tie Game
  • Game still in progress

Mapping input to Tic Tac Toe board

Each character in your input represents a space on a Tic Tac Toe board. The mapping is as follows (by index):

The Characters

Your input string will consist of the following characters:

  • x = Player X occupies this space
  • o = Player O occupies this space
  • - = No player occupies this space
So for example, your inputs will look like: "--xxoo---". And that would map to the following Tic Tac Toe board: For the purposes of this exercise, you don't have to validate your input. You can assume that it will always be nine (9) characters, and that it will only contain some combination of the "x", "o", and "-" characters.

Sample Inputs & Outputs

To help test your program, here are some sample inputs and what your program should output:

Input Output
xxx-o-o-o X wins
xo-ox-o-x X wins
x-o-o-oxx O wins
x-xooo-x- O wins
ooxxxooxx Tie Game
oxooxoxox Tie Game
--------- In Progress
